## Projects

### 1. AI News Collector Bot
Fetches financial news from RSS feeds and analyzes sentiment using Claude AI.
- Scores articles from -1.0 (bearish) to +1.0 (bullish)
- Generates BUY / SELL / HOLD trading signals
- Identifies key market themes automatically

```bash
pip install requests
python news_collector.py
```

---

### 2. Automated PDF Report Generator
Analyzes US stocks and generates a professional PDF report.
- Covers RSI, MA20, MA50, 52-week high/low
- Color-coded BUY / SELL / HOLD signals
- Clean multi-page PDF with executive summary

```bash
pip install yfinance pandas fpdf2
python report_generator.py
```

---

### 3. Web Scraper Dashboard
A real-time dashboard UI for monitoring scraping jobs and results.
- Live job status tracking
- Success rate per source
- Activity log with timestamps
- Open `scraper_dashboard.html` in any browser — no server needed
